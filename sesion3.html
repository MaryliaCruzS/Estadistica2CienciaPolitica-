üéØ ¬øPara qu√© sirve la Regresi√≥n Log√≠stica Binaria?


La regresi√≥n log√≠stica binaria se utiliza cuando la variable dependiente es dicot√≥mica, es decir, solo toma dos valores posibles (por ejemplo: 1 = s√≠, 0 = no).

Este modelo permite estimar la probabilidad de que ocurra un evento, como ser voluntario, votar, enfermarse, etc., en funci√≥n de variables predictoras (edad, g√©nero, ingreso, etc.).

‚úÖ ¬øQu√© representa el valor 1** en una regresi√≥n log√≠stica?**

En una regresi√≥n log√≠stica binaria, la variable dependiente solo puede tomar dos valores: 0 y 1.

  • El valor 1 representa el evento que queremos predecir o explicar.
  • El valor 0 representa la ausencia de ese evento.

El modelo se enfoca en estimar la siguiente probabilidad:

\[ \text{Prob}(Y = 1 \mid X) = p \]

Es decir, el modelo calcula la probabilidad de que ocurra el evento codificado como 1, dado un conjunto de variables explicativas.

‚öôÔ∏è ¬øC√≥mo funciona?

La regresi√≥n log√≠stica modela la probabilidad de que ocurra un evento usando la funci√≥n log√≠stica (sigmoide). El modelo estima el logaritmo de los odds como una funci√≥n lineal:

\[ \log\left(\frac{p_i}{1 - p_i}\right) = \beta_0 + \beta_1 X_{1i} + \beta_2 X_{2i} + \dots + \beta_k X_{ki} \]

donde \(p_i\) es la probabilidad de que el evento ocurra para el individuo \(i\).

Luego se transforma a escala de probabilidad con:

\[ p_i = \frac{e^{\beta_0 + \beta_1 X_{1i} + \dots + \beta_k X_{ki}}}{1 + e^{\beta_0 + \beta_1 X_{1i} + \dots + \beta_k X_{ki}}} \]


üß™ ¬øCu√°ndo usarla?


Usa regresi√≥n log√≠stica binaria cuando:

  • La variable dependiente es binaria (0/1)
  • Quieres estimar probabilidades
  • Los predictores pueden ser categ√≥ricos o num√©ricos
  • No se cumplen los supuestos de normalidad o homocedasticidad de la regresi√≥n lineal

Si hay m√°s de dos categor√≠as en la variable dependiente, deber√≠as usar regresi√≥n log√≠stica multinomial (multinom()).

  • LogLik (Log-likelihood): mayor valor indica mejor ajuste.

Se refiere a la habilidad del modelo para distinguir entre los casos 0 y 1, es decir, entre quienes s√≠ y no presentan el evento.
